- **Scenario: Department & Unit Selection**
    - สำหรับตัวแทนหน่วยงานอื่น ระบบต้องทำการ **Default** ค่าหน่วยงานให้ตรงกับต้นสังกัดของผู้ใช้งานโดยอัตโนมัติ.
    - ระบบต้องทำการ **Disable (Read-only)** ช่องเลือกหน่วยงาน เพื่อป้องกันไม่ให้ผู้ใช้งานแก้ไขหรือเลือกหน่วยงานอื่นที่ตนเองไม่ได้สังกัด.
    - สำหรับตัวแทนหน่วยงานอื่น ระบบต้องแสดงเฉพาะรายชื่อ **"ฝ่าย"** ที่ผู้ใช้งานคนนั้นสังกัดอยู่เท่านั้น
        - หากสังกัดอยู่เพียง **ฝ่ายเดียว** ระบบต้องทำการ **Default** เลือกฝ่ายนั้นให้โดยอัตโนมัติ และ**ล็อก (Disable)** ช่องเลือกฝ่ายเพื่อไม่ให้แก้ไขได้
        - หากสังกัดอยู่ **หลายฝ่าย** ภายใต้หน่วยงานเดียวกัน ระบบต้องอนุญาตให้ผู้ใช้งานสามารถ **เลือก (Select)** ฝ่ายที่ต้องการจากรายการที่สังกัดอยู่ได้
    - สำหรับพี่เหมี่ยว/พี่เมฆ ระบบต้องอนุญาตให้สามารถเลือก **หน่วยงาน**ใดก็ได้ในระบบผ่านตัวเลือกแบบ **Select** เพื่ออำนวยความสะดวกในการนำเข้าโครงการแทนหน่วยงานอื่น
- **Scenario: Budget Selection**
    - ผู้ใช้สามารถเลือก **แผนงบประมาณ** ได้หลายรายการ (Multi-select) โดยระบบต้องแสดงข้อมูล ชื่อกิจกรรม, ชื่อศูนย์ต้นทุน และงบประมาณ.
    - ระบบต้องคำนวณและเติมค่า **วงเงินงบประมาณ** ให้โดยอัตโนมัติ (Default) ตามผลรวมของแผนงบประมาณที่เลือก
    - หากยอดเงินที่ระบุ **น้อยกว่า** ค่า Default (ผลรวมจากแผนงบประมาณที่เลือก) ระบบต้องแสดง **ข้อความแจ้งเตือน (Warning)**
- **Scenario: Dynamic Selection**
    - ตัวเลือก **ฝ่าย** ต้องแสดงเฉพาะ Unit ที่ผู้ใช้สังกัดอยู่
    - ตัวเลือก **แผนงบประมาณ** ต้องแสดงเฉพาะข้อมูลของหน่วยงานนั้น และต้องยังไม่ถูกเลือกไปสร้างโครงการอื่น
- **Scenario: Date Warning**
    - หาก **วันที่ส่งมอบ** น้อยกว่าเกณฑ์ขั้นต่ำ (Min Days) ของวิธีการจัดหานั้น ระบบต้องแสดง **Alert Warning** เพื่อยืนยันวันที่ส่งมอบ

### ทั้ง 3 ทาง **หลังกดยืนยันการนำเข้า**

- **Scenario: Data Assignment**
    - ระบบต้อง **Gen เลขรับ** ในรูปแบบ `2569/XXXXX` อัตโนมัติ
    - ระบบต้อง **คำนวณความเร่งด่วน** ตามเกณฑ์ของระบบ (ปกติ, ด่วน, ด่วนพิเศษ)
    - โครงการใหม่ต้องไปปรากฏที่หน้า **โครงการทั้งหมด** ในสถานะ **"ยังไม่ได้มอบหมาย"**
    - โครงการต้องถูกจัดเข้า **Pool งาน** ตามกลุ่มงานที่รับผิดชอบ

**Scenario: Navigation Post-Success**

- **ผู้ใช้ภายนอก**: หลังสร้างเสร็จ หากต้องการนำเข้าโครงการเพิ่มเติมให้กลับไปที่หน้า **"กรอกฟอร์ม"**
- **สารบรรณ/พี่เมฆ**: หลังนำเข้าเสร็จ หากต้องการสร้างโครงการเพิ่มเติมให้กลับไปที่หน้า **"เลือกวิธีนำเข้า"**
- มีปุ่มกลับไปหน้าโครงการทั้งหมด

เข้าถึงหน้าได้เฉพาะ **สารบรรณ/พี่เมฆ, ตัวแทนหน่วยงานเท่านั้น**
